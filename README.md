# Aurevia

Aurevia is a local-first desktop productivity app for PC.

It is designed to combine:
- a Notion-like planning and tracking experience
- a lightweight calendar for deadlines and scheduling
- a visual planning canvas with connected boxes
- a simple design workspace for web layout exploration

Version 1 is focused on being:
- local-only
- beginner-friendly
- modular
- open-source

## Vision

Aurevia is meant to help users plan projects, track tasks, visualize workflows, and gradually connect planning, calendar, and design work inside one desktop app.

The long-term goal is not to clone Notion or Figma fully.
The goal is to create a clean, practical workspace that feels easy to use and grows in phases.

## Getting Started

### Prerequisites

- Node.js
- npm

### Install

```bash
npm install
```

### Run

```bash
npm start
```

If the project later uses a different dev command, update this section accordingly.

## Development Approach

This project is being built in phases:

1. Phase 1 - foundation setup
2. Phase 2 - app shell
3. Phase 3 - home and project creation flow
4. Phase 4 - local storage and recent files
5. Phase 5 - planning canvas
6. Phase 6 - calendar
7. Phase 7 - workflow linking
8. Phase 8 - basic design workspace
